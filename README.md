## 🌿 About the Project

AI Biology Tutor is an educational desktop chatbot that provides quick and understandable explanations for a wide range of Biology topics.

Instead of navigating through textbooks or notes, students can simply ask a question and instantly receive an answer in a friendly chat environment.

The project combines a clean graphical interface with lightweight text-matching techniques, making it ideal for learning purposes and beginner-friendly AI projects.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/ai-biology-tutor.git
```

Navigate to the project folder:

```bash
cd ai-biology-tutor
```

Run the application:

```bash
python ai_biology_tutor.py
```

---

## 🧠 How It Works

The chatbot follows a straightforward workflow:

```
User Question
       │
       ▼
Text Normalization
       │
       ▼
Similarity Matching
       │
       ▼
Best Match Selection
       │
       ▼
Biology Response
```

If no suitable match is found, the chatbot politely encourages the user to ask another Biology-related question.
